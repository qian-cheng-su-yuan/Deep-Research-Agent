项目运行步骤

按顺序准备配置、启动服务、填写主题并生成报告，完整流程可直接用于项目交付验收。

  1. 安装依赖并按需复制 `.env.example` 为 `.env`。
  2. 无 Key 时使用本地模式，接入 DeepSeek / Qwen 后切换真实模型。
  3. 启动 FastAPI 服务并确认页面显示服务状态。
  4. 填写研究主题、模型提供商和输出目录后生成报告。
  5. 检查审核状态、报告路径和 Markdown 内容，再复制或下载交付文件。
